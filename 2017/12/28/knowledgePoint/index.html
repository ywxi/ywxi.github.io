前端知识点记录

  • web

1、中文版chrome默认最小字号12px，如何设置小于12px的字体？

2、配置hosts文件

在浏览器中输入www.baidu.com之后，系统会先检查本地hosts文件是否存在域名映射，如果存在，则域名解析到此完成，在本地开发时大都是这么做。

3、建立测试数据

项目下建立一个mock文件夹，里面直接新建json文件用来模拟请求回来的数据，比如goodslist.json。注：json文件里不能写注释，不能有多余的逗号，key需要用双引号包起来。

4、shell 终端

command+t 可以打开一个新的shell终端窗口，mac可以连续打开多个终端窗口。

终端里可以直接手动拖入文件，他会自动识别出文件路径，而不用依次手动输入路径。

5、pm2

用pm2启动node程序，可以监控请求日志，可在一个终端窗口下启动多个进程，可在同一窗口下做更多的操作。

mongoose是对mongodb的封装及扩展。

res.end()输出的是字符串，res.json()可直接输出json串。

websocket用的是ws协议。

6、node

1、基于chrome v8引擎 2、单线程 3、用js开发后端代码 4、非阻塞I/O

Express框架：通过express-generator生成器快速构建一个项目 ，构建完成后先安装依赖，在运行项目。

bin文件夹里放的是可执行文件。

vue里：assets 和static都是用来存放静态资源的，assets更加偏向于放组件的资源。页面上的资源放在static.(static里的图片不会转变，它会以src的方式加载，一般都相对来说较大，而assets里的图片相对较小，一般都会打包到页面，不会以src的方式加载，一般会转成base64.)

7、js获取UA(user agent)用户代理

通过navigator对象来获取：

navigator.appName;//返回浏览器的的名称

navigator.appCodeName;//返回浏览器的代码名

navigator.appVersion;//返回浏览器的平台和版本信息

userAgent 返回由客户机发送服务器的user-agent头部的值

platform 返回运行浏览器的操作系统平台

cookieEnabled 返回指明浏览器中是否启用cookie的布尔值

其他参数可以参照navigator对象

9、jquery的ajax返回的是promise对象吗

不是，jquery的ajax返回的是deferred对象，通过Promise.resolve()方法将其转换为promise对象

10、transform与transition区别

transform:2D/3D转换； transition:过渡； 两者可单独使用

transform 属性向元素应用 2D 或 3D 转换。该属性允许我们对元素进行旋转、缩放、移动或倾斜。它并不是一个动态属性，它是一个静态属性，注意它只是改变，并不是一个过程，所以该属性只对元素进行变换，不会产生过渡效果。它只是针对css的各种特殊改变，完全可以看作是与width，height，position等同类的属性。

transition可以实现动态效果，实际上是一定时间之内，一组css属性变换到另一组属性的动画展示过程。

常用的四个属性：

transition-property:all; 设置过渡效果的css属性名称

transition-duration:过渡完成时间 transition-delay:是否延迟过渡

transition-timing-function：规定速度效果的速度曲线

一般来说transition属性应用到最初的样式里，而不是放在结束的样式里，即定义动画开始之前的元素外观的样式。只需要给元素设置一次transition，浏览器就会负责以动画展示从一个样式到另一个样式，再返回最初样式的变化过程。

11、git status

git status -s 以精简的方式显示文件状态 -s表示short,-s的输出标记会有两列，第一列是对staging区域而言，第二列是对working目录而言。

工作区有一个隐藏目录.git，这个不算工作区，而是git的版本库。stage暂存区也存在于.git文件夹中。

12、gitHub用T键搜索

github搜索某个库时，如libuv，在搜出来的结果中点进去，再直接按’’T键’’进行搜索，可以搜索包含搜索内容的所有文件。

13、unix时间戳

Timestamp:现在至1970年的毫秒数。

unix时间戳转换成标准时间：new Date(Timestamp)=>标准时间

14、post请求

发送post请求一定要设置contentType，否则后台收不到请求体内容。发送post请求浏览器将参数自动格式化为key-value形式，如果只传vlaue，调试工具后台请求体参数你会发现’value:’,value当成了键，值为空。默认给转成了k-v形式。一定要规范。

15、js实现千位分隔符—tolocaleString()

数字的千位分隔符：三位一个逗号间隔。实现方法：(Number).toLocaleString()
